class com.yagp.structs.GifFrame

Available on all platforms

Single frame of gif file.

Instance Fields

var data:BitmapData

Output BitmapData of frame

var delay:Int

Delay, before start rendering next frame in milliseconds. Note: Most browsers forces 0 and 1 (0/10 milliseconds) delay to 10 (100ms). To disable this behaviour add a yagp_accurate_delay define.

var disposalMethod:DisposalMethod

Disposal method of frame

var height:Int

Height of frame

var transparentIndex:Int

Index of transparent color in this frame. If frame has no transparency, value is -1

var userInput:Bool

Is this frame must wait input from user, before render next frame?

var width:Int

Width of frame

var x:Int

X coordinate of frame on logical screen

var y:Int

Y coordinate of frame on logical screen
